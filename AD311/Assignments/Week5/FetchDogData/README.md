# Fetching Dog Data with JavaScript
This project is for experimenting with fetching data from dogapi.dog using JavaScript.

To try out the project, run **npm start** in a console at the root folder.

The functions will be output to the console.

## Functions
- *getDogData()*: Gets a list of data about dogs.
- *getDogDataById(id)*: Gets data of a specific dog using an id.
- *getDogFacts()*: Gets a list of dog facts.
- *getDogGroups()*: Gets a list of groups of dogs.
