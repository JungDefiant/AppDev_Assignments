# Dog Facts API v1 Replica with NodeJS and ExpressJS
This project is for creating an API server that returns facts about dogs.

To try out the project, follow the steps below.
1. Run **npm start** in a console.
2. Open a browser and navigate to localhost:3000/facts. You should see an output of dog facts in an array.
3. Navigate to localhost:3000/facts/5. You should see an array of 5 dog facts.

## Routes
- */facts*: Gets the entire list of dog facts.
- */faces/{number}*: Gets specific number of dog facts.
