# Build a Basic Express NodeJS Server

This project is a basic Express NodeJS server.

Run the project using **npm start**, then navigate to *localhost:3000* on your browser.

## Routes
- */about*: About page
- */foo*: Randomized output
- */user*: User page
- */username*: User page
- */user/<name>*: User page with specific name
- */userquery?username=<name>*: User page with specific name using query
